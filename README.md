# DigiTools

DigiTools is a modern, responsive all-in-one digital tools platform where users can explore products, view pricing plans, and manage their selections through an interactive cart system. It combines clean UI, smooth interactions, and practical features to deliver a seamless and engaging user experience, with a strong focus on simplicity, performance, and visually appealing design.

---

## ⚙️ Technologies Used

*⚛️ React.js
*🎨 Tailwind CSS & DaisyUI
*🟨 JavaScript (ES6+)
*🔔 React Toastify (NPM Package)
*📦 JSON (for product data)

---

## ✨ Features

### 🛒 Smart Cart Management

* Add and remove products dynamically
* Real-time cart updates in the navbar
* Smooth and intuitive user interaction

### 💎 Modern UI & Responsive Design

* Clean and minimal design using modern typography
* Fully responsive across devices
* Eye-catching sections like Hero, Stats, and Pricing

### 📦 Dynamic Product & Pricing System

* Interactive product listing
* Structured pricing plans with reusable components
* Clear and user-friendly layout for decision making
